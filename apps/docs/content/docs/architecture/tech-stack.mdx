## Core

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js 16, React 19, Tailwind CSS v4 |
| Backend | Motia (iii engine), TypeScript step handlers |
| Database | Supabase (PostgreSQL) with Row-Level Security |
| Auth | Better Auth |
| Queue | Redis (BullMQ via Motia) |
| Real-time | Yjs (CRDT), WebSocket streams |
| Monorepo | Turborepo, Bun |

## State Management

| Pattern | When to use |
|---------|-------------|
| SWR CRUD Factory | Server data with simple CRUD (recommended) |
| Raw SWR + Cache Orchestrator | Complex server queries, pagination |
| Zustand | Complex client-side state |
| Context API | Component-tree scoped state |
| useState | Local component state |
